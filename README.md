# 🌐 Niaz Hussain Abbasi – Personal Landing Page

A modern, responsive personal landing page built to showcase the professional profile, technical expertise, and personal journey of **Niaz Hussain Abbasi** — a multi-skilled civil engineering professional and community-oriented individual.

---

## 🚀 Overview

This project is a high-quality, production-ready landing page designed with a strong focus on:

- Clean UI/UX
- Performance
- Accessibility
- Professional storytelling

It highlights both **technical expertise** (engineering, construction, documentation) and **human elements** (resilience, creativity, and social impact).

---

## ✨ Features

- ⚡ Modern React + Tailwind CSS architecture
- 📱 Fully responsive (mobile-first design)
- 🎯 Clear visual hierarchy and UX-focused layout
- 🧩 Component-based structure (scalable & maintainable)
- 🎨 Clean and minimal design system
- 🧠 Optimized for readability and engagement
- 🚀 Fast performance and lightweight build

---

## 🧱 Tech Stack

- **Frontend:** React
- **Styling:** Tailwind CSS
- **Build Tool:** Vite / Next.js (depending on setup)
- **Animations:** Framer Motion (optional)
- **Best Practices:**
  - Vercel React Best Practices
  - Anthropic Frontend Design Skills

---

## 📂 Project Structure

```
/src
  /components     → Reusable UI components
  /sections       → Page sections (Hero, About, etc.)
  /assets         → Images & static files
  /styles         → Global styles/config
  /utils          → Helper functions
```

---

## 🧩 Sections Included

- **Hero** – First impression with strong positioning
- **About** – Professional summary + personal journey
- **Experience** – Timeline of engineering roles
- **Skills** – Technical & soft skills
- **Work Highlights** – Practical project exposure
- **Personal Side** – Poetry, social work, content creation
- **Contact** – Easy communication access

## 🛠️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/niaz-abbasi-landing-page.git
cd niaz-abbasi-landing-page
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```
