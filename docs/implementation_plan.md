# Implementation Plan (Refined)

## Architecture

- React (Vite or Next.js)
- Tailwind CSS
- Component-first structure

---

## Component Strategy

Each section = isolated + reusable

---

## Priority Order

1. Layout skeleton
2. Hero (critical)
3. Sections (static)
4. Styling system
5. Animations (last)

---

## Performance Rules

- No heavy libraries unless needed
- Lazy load images
- Avoid unnecessary state

---

## Design Constraints

- Max width: 1200px
- Mobile-first
- Consistent spacing scale

---

## Animation Rules

- Only:
  - Fade-in
  - Slight translate
- Avoid:
  - Overuse of motion

---
